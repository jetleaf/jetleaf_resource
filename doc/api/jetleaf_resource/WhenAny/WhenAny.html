Creates a composite OR condition that evaluates to true if either left or right evaluates to true.

A composite ResourceCondition that evaluates to true if either the left or right condition evaluates to true.

This condition allows flexible conditional logic for resource operations, enabling a resource to be applied when at least one of multiple criteria is satisfied.

Example

@RateLimit(condition: WhenAny(WhenExpr('#user.isAdmin'), WhenExpr('#user.isModerator')))
Future<void> performAction(User user) async {
  // Action allowed for admin or moderator users
}

In this example, rate limiting is applied only if the user is either an admin or a moderator.
