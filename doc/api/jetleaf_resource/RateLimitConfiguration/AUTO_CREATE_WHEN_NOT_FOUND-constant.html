AUTO_CREATE_WHEN_NOT_FOUND constant

String const AUTO_CREATE_WHEN_NOT_FOUND

Configuration property key that controls automatic rate-limit creation.

When set to true (e.g., in application configuration), JetLeaf will automatically create a new rate-limit instance when a requested rate-limit is not found in any registered RateLimitManager or RateLimitStorage.

When false, missing rateLimits might trigger a NoRateLimitFoundException.

Defaults to false if unspecified.
