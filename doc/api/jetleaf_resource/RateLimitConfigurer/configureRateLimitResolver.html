Configures and registers RateLimitResolver implementations.

The resolver defines how the framework determines which RateLimitStorage instances apply to a given RateLimit annotation.

Implementations may register multiple resolvers to support advanced scenarios, such as dynamic context-based routing or composite resolution strategies.

Parameters

  • registry: The registry used to add resolver instances.

Example

registry.addResolver(DefaultRateLimitResolver());
