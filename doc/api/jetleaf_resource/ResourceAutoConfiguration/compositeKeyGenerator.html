KeyGenerator compositeKeyGenerator()

Registers the CompositeKeyGenerator as the active KeyGenerator when the configuration property jetleaf.resource.key-generator is set to "composite" or is missing.

The composite generator coordinates multiple registered KeyGenerators (including conditional generators) into a deterministic chain, ensuring flexible and extensible key resolution.
