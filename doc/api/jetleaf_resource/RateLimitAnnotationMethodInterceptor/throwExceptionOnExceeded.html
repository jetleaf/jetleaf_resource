throwExceptionOnExceeded property

bool throwExceptionOnExceeded
getter/setter pair

Indicates whether the rate-limit operation should throw a RateLimitExceededException when a limit is exceeded.

  • true: Exceptions are thrown on exceed events.
  • false: The operation records the denial and returns control silently, allowing the caller to inspect context state via allowed or retryAfter.

This flag is typically driven by the throwOnExceeded property of the @RateLimit annotation or global configuration.
